# 予防検査プロトコル & 受診履歴ダッシュボード

> **目的**：日本人死因TOP5（がん・心疾患・脳血管疾患）に対する予防検査を一覧化し、
> 「いつ受けるべきか」「過去いつ受けたか」「次回いつか」を一目で把握できるようにする。
>
> **使い方**：社長が過去履歴をこのファイルに入力 → AIが次回予定を算出 → 期限超過があればGmail通知。
> 実装後は `exam_schedule.gsheet` に同期され、自動更新される。

## 2. 部位別プロトコル詳細（医学的根拠と判断基準）

### 2.1 胃がん（男性がん死因 11.4%、第3位）
- **ピロリ菌検査が最重要**：日本の胃がんの約99%はピロリ菌感染が背景
- **陰性確定 → 胃がんリスクは大幅に低下**、以後内視鏡頻度を下げてよい
- 陽性 → 除菌治療＋以後1〜2年に1回内視鏡
- 検査方法：尿素呼気試験 / 血中抗体 / 便中抗原
- **アクション**：未実施なら最優先で実施

### 2.2 大腸がん（男性がん死因 12.6%、第2位）
- 便潜血：簡便・年1回・健診で実施可
- 大腸内視鏡：陽性or50歳以上は5年に1回が標準
- ポリープ切除歴あり → 1〜3年に短縮
- **加藤社長へ**：50歳前後なら一度ベースライン内視鏡を推奨

### 2.3 肺がん（男性がん死因 23.9%、第1位）
- 最大リスク：喫煙・受動喫煙
- **喫煙歴なし**でも年1回胸部X線、リスク高ければ低線量CT年1
- **アクション**：喫煙状況を `master_health` に記録、それに応じて頻度調整

### 2.4 膵臓がん（男性がん死因 9.0%、第4位）
- 早期発見が困難・予後不良 → スクリーニング重要
- リスク要因：糖尿病、家族歴、慢性膵炎、喫煙、肥満
- 腹部エコー＋CA19-9を年1で組み合わせるのが現実的
- **アクション**：糖尿病マーカー（HbA1c）の悪化があれば膵臓検査強化

### 2.5 肝臓がん（男性がん死因 6.9%、第5位）
- 主因：肝炎ウイルス（HBV/HCV）、アルコール、脂肪肝
- **肝炎ウイルス検査は一生に1回**で十分（陰性なら）
- 陰性 → 腹部エコー＋AFPを年1
- **アクション**：未検査なら早急に実施

### 2.6 心疾患・脳血管疾患
- 共通基盤：高血圧・脂質異常・糖代謝・喫煙・肥満
- 健診で血圧・LDL・HDL・中性脂肪・HbA1c・空腹時血糖を年1モニタ
- 動脈硬化進行は頸動脈エコーで可視化（数年に1回）
- 脳ドック（MRI/MRA）は40〜50代で初回実施 → 異常なければ3〜5年間隔

### 2.7 男性ホルモン（テストステロン）
- 既に専門クリニック受診中
- 半年〜年1回の継続モニタ推奨
- **専用分析テンプレート**で総テストステロン・遊離テストステロン・関連項目（DHEA-S等）の推移を可視化
- 詳細は `analysis_templates/testosterone_review.md`（フェーズ1Aで作成）

## 4. 次回予定の自動算出ルール

| 頻度区分 | ロジック |
|---|---|
| 年1回 | 最終受診日 + 12ヶ月 |
| 半年に1回 | 最終受診日 + 6ヶ月 |
| 1〜2年 | 最終受診日 + 18ヶ月（中央値） |
| 3〜5年 | 最終受診日 + 4年（中央値） |
| 5年に1回 | 最終受診日 + 5年 |
| 一生に1回（陰性） | 結果が陰性なら次回不要、陽性なら除菌後1年で胃内視鏡 |

**期限超過判定**：次回予定日 < 今日 → ⚠️ ステータス
**早期警告**：次回予定日まで30日以内 → 🟡 「もうすぐ予定日」

## 5. 月次/四半期での運用

### 月次（自動）
- GASが本ファイル（または `exam_schedule.gsheet`）をスキャン
- ⚠️ 期限超過がある検査をリスト化
- Gmail通知＋秘書朝レポートに表示

### 四半期（手動レビュー）
- ダッシュボード全体を社長と一緒にレビュー
- 直近の数値変化（脂質悪化など）に応じて頻度を調整
- 例：HbA1c上昇 → 膵臓関連検査を半年に短縮
