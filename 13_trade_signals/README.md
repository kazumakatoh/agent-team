# トレードシグナル ダッシュボード

MEXC取引高上位銘柄を移動平均線ロジックでスクリーニングし、トレードシグナルを表示するシステム。

## セットアップ

```bash
cd 13_trade_signals
pip install -r requirements.txt
```

## 起動方法

```bash
streamlit run app.py
```

ブラウザで `http://localhost:8501` が自動で開きます。

## ファイル構成

| ファイル | 役割 |
|---|---|
| `app.py` | Streamlitダッシュボード（メイン画面） |
| `mexc_api.py` | MEXC APIからデータ取得 |
| `analyzer.py` | 移動平均線でシグナル判定 |
| `config.py` | 設定値（銘柄数、MA期間、更新間隔など） |

## 機能

- MEXC取引高上位50銘柄を自動取得
- 日足 / 4時間足を切り替え可能
- MA5/10/30/50/100でシグナル判定
- ローソク足＋MA線チャート表示
- シグナルフィルタ機能
- 10分ごと自動更新

## シグナル一覧

| シグナル | 意味 | 条件（暫定） |
|---|---|---|
| 🟢 強い上昇相場 | ロング推奨 | パーフェクトオーダー＋価格>MA5 |
| 🔵 上昇相場 | ロング推奨 | MA5>10>30 |
| 🟡 上昇の兆し | ロング準備 | MA5がMA10をGC |
| 🟠 下落の兆し | ショート準備 | MA5がMA10をDC |
| 🔴 下落相場 | ショート推奨 | MA5<10<30 |
| ⚫ 強い下落相場 | ショート推奨 | 逆パーフェクトオーダー＋価格<MA5 |
| ⚪ 横ばい | 様子見 | 上記いずれにも該当しない |
