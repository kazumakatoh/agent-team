# キャッシュフロー管理表 運用マニュアル

## 概要

株式会社LEVEL1の資金繰りを管理するためのExcelベースのキャッシュフロー管理表。
3つの銀行口座を横断して、日次・月次で資金状況を把握し、キャッシュアウトを防ぐ。

---

## 管理対象口座（3口座）

| 口座名 | 略称 | 用途 |
|--------|------|------|
| PayPay銀行 ビジネス営業部 | CF005（PayPay 005） | 法人メイン口座 |
| PayPay銀行 はやぶさ支店 | CF003（PayPay 003） | 法人サブ口座 |
| 西武信用金庫 阿佐ヶ谷支店 | 西武信金 | 融資・返済用口座 |

---

## シート構成

| シート名 | 内容 |
|----------|------|
| **Daily** | 3口座の日次入出金記録（実績＋予定） |
| **CF005** | PayPay 005の月別集計 |
| **CF003** | PayPay 003の月別集計 |
| **西武信金** | 西武信用金庫の月別集計 |

| **データ** | マスタデータ・参照用 |
| **経費一覧** | 経費の内訳管理 |

## 3. 月別シート（3口座合算サマリー）

### 目的
3口座を合算した全体像で、会社全体の資金状況を把握する。

### 構成

| 列 | 内容 |
|----|------|
| A | 年度 |
| B | 入金（3口座合計） |
| C | 出金（3口座合計） |
| D | 差（入金 − 出金） |
| E | PayPay銀行 法人口座 残高 |
| F | PayPay銀行 個人口座 残高 |
| G | 西武信用金庫 残高 |
| H | 残高（3口座合計） |

### 計算式の例
- `D16` = `'CF005'!AX5 + 'CF003'!AA5`（各口座シートの出金を合算）

## 月次運用フロー

### 月初（1日〜3日）
1. 前月のDailyシートの入出金実績を最終確定
2. 口座別月次シート（CF005/CF003/西武信金）に前月データが正しく反映されているか確認
3. 月別シートの合算値を確認

### 月中（随時）
4. Dailyシートに入出金実績を都度記録（上書き更新）
5. 今後の入出金予定を手入力で追記
6. 列Aの合計残高を確認し、キャッシュアウトリスクがないかチェック

### 月末
7. 月末時点の各口座残高を確認
8. 翌月の入出金予定を事前入力

## アラート基準（キャッシュアウト防止）

| レベル | 条件 | アクション |
|--------|------|------------|
| 🔴 危険 | 3口座合計残高が1,000万円を下回る見込み | 社長に即報告・資金移動検討 |
| 🟡 注意 | 3口座合計残高が2,000万円を下回る見込み | 入出金予定の再確認・支払い調整検討 |
| 🟢 正常 | 3口座合計残高が2,000万円以上 | 通常運用 |

※2026年3月末時点の残高は約5,300万円で正常水準。

---

## 注意事項

1. **Amazon売上の入金タイミング**：2週間サイクルのため、月によって1回or2回の入金がある
2. **JALカード引き落とし**：170万円と大口のため、引き落とし月の残高に注意
3. **融資返済**：西武信用金庫から毎月自動引き落とし（複数本あり、合計約50万円/月）
4. **資金移動**：口座間の資金移動は入金・出金の両方に計上されるため、実質的なキャッシュフローには影響なし
5. **金額の桁確認**：日本円の桁読み間違いに常に注意する
