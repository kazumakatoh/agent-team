# 資金繰り表 × MoneyForward 勘定科目マッピング

## 概要
資金繰り表の各項目が、MoneyForward クラウド会計のどの項目から取得できるかの対応表。

## フォーマット
- 単位：千円
- 期間：年度別シート（決算月=2月 → 3月〜翌2月）
- ベース：日本政策金融公庫 提供フォーマット

## MFでの確認方法

### 月次推移で取れる項目
MF会計 > **レポート > 月次推移** で以下を確認：
- 損益計算書の月次推移 → 売上高、販管費、営業外損益
- 貸借対照表の月次推移 → 現金預金残高、商品残高

### 仕訳帳から取る項目
MF会計 > **仕訳帳** で勘定科目を絞り込み：
- 売掛金回収：「売掛金」で検索 → 貸方が売掛金の仕訳 = 回収
- 買掛金支払：「買掛金」で検索 → 借方が買掛金の仕訳 = 支払
- 借入金：「長期借入金」で検索 → 貸方=借入実行、借方=返済
