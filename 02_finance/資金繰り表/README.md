# 資金繰り表（キャッシュフロー計算書）

## 概要
日本政策金融公庫提供フォーマットに基づく月別資金繰り表。
MoneyForward クラウド会計APIと連携し、自動生成が可能。

## ファイル構成
| ファイル | 内容 |
|---|---|
| `資金繰り表_2025.csv` | 第7期（2024年3月〜2025年2月）実績 |
| `資金繰り表_2026.csv` | 第8期（2025年3月〜2026年2月）テンプレート |
| `mf_cashflow.py` | MoneyForward API連携スクリプト |
| `config.json` | API認証情報・勘定科目マッピング |

## 単位
千円

## 構造（金融公庫フォーマット）
```
前月繰越金 (A)
├─ 経常収入 計 (B) = 現金売上 + 売掛金回収 + その他
├─ 経常支出 計 (C) = 現金仕入 + 買掛金支払 + 人件費 + 商品棚卸高 + 諸経費
├─ 差引過不足 (D) = (B) - (C)
├─ 経常外収支 計 (E) = 経常外収入 - 経常外支出
├─ 財務収支 計 (F) = 借入 - 返済
└─ 翌月繰越金 (G) = (A) + (D) + (E) + (F)
```

## MoneyForward API連携

### セットアップ手順

#### 1. MoneyForward APIアプリを作成
1. [MoneyForward クラウド会計](https://accounting.moneyforward.com) にログイン
2. **API連携 > アプリ管理** から新規アプリを作成
3. `client_id` と `client_secret` を取得

#### 2. config.json に認証情報を設定
```json
{
  "moneyforward": {
    "client_id": "取得したclient_id",
    "client_secret": "取得したclient_secret",
    ...
  }
}
```

#### 3. OAuth2認証を実行
```bash
python mf_cashflow.py --period 8 --auth
```
ブラウザで認証 → 表示される認可コードを入力 → トークンが自動保存されます。

#### 4. 資金繰り表を生成
```bash
# 第8期の資金繰り表を生成
python mf_cashflow.py --period 8

# 第7期の資金繰り表を再生成（MFデータで更新）
python mf_cashflow.py --period 7
```

### データ取得ロジック

| 資金繰り表の項目 | MF APIデータソース | 算出方法 |
|---|---|---|
| 今期売上 | PL（売上高） | 月次発生額 |
| 前月繰越金 | BS（現金・預金） | 前月末残高 |
| 売掛金回収 | BS（売掛金）+ PL（売上） | 前月AR + 当月売上 - 当月AR |
| 買掛金支払 | BS（買掛金） | 前月AP - 当月AP（減少分） |
| 人件費 | PL（役員報酬・給料等） | 月次発生額 |
| 商品棚卸高 | BS（商品） | 月次在庫増減 |
| 諸経費 | PL（販管費） | 月次発生額 |
| 経常外収支 | PL（営業外損益） | 月次発生額 |
| 借入・返済 | 仕訳データ | 借入金勘定の貸方/借方 |

### 勘定科目マッピング
`config.json` の `account_mapping` で、MFの勘定科目名と資金繰り表の項目を対応付けています。
MFで使用している勘定科目名が異なる場合は、ここを編集してください。

### 注意事項
- `config.json` には認証トークンが含まれるため **Git管理しない**（.gitignoreに追加済み）
- アクセストークンの有効期限切れ時は自動リフレッシュされます
- 外部依存ライブラリなし（Python標準ライブラリのみ）
