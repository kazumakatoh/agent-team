# 資金繰り表

日本政策金融公庫フォーマットに基づく月別資金繰り表。
数値はMoneyForward クラウド会計から取得。

## ファイル構成
| ファイル | 内容 |
|---|---|
| `資金繰り表_2025.csv` | 第7期（2024/3〜2025/2）実績 |
| `資金繰り表_2026.csv` | 第8期（2025/3〜2026/2） |
| `MFマッピング.md` | 各項目 ↔ MF会計の勘定科目 対応表 |

## 運用
- **スプレッドシート**で管理（年度別シート）
- MF会計の月次推移・仕訳帳から該当数値を転記
- 合計・繰越金はスプシの数式で自動計算
