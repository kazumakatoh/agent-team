## ⑧ 📖 FX用語集

| 用語 | 読み | 一言定義 | 計算式/詳細 | 関連用語 |
|---|---|---|---|---|
| PF ★ | プロフィットファクター | 総利益÷総損失。1超えで黒字 | Gross Profit / Gross Loss | RR比 |
| RR比 ★ | リスクリワード比 | 平均利益÷平均損失 | avg Profit / avg Loss | PF, 期待値 |
| pips | ピップス | 価格変動の最小単位 | ゴールド：$0.01=1pip | ロット |
| ロット | - | 取引量の単位 | 1lot=10万通貨、ゴールド1lot=100oz | レバレッジ |
| レバレッジ | - | 証拠金の何倍取引可能か | BigBoss最大2222倍 | 証拠金維持率 |
| 証拠金維持率 | - | 口座の健全度。100%割れで強制決済 | (有効証拠金/必要証拠金)×100% | レバレッジ |
| スプレッド | - | 買値と売値の差。実質手数料 | Ask - Bid | コミッション |
| スワップ | - | 通貨間金利差の日割り | 長期保有時 | - |
| ドローダウン | DD | 高値からの最大下落率 | (最大残高-最小残高)/最大残高 | 最大連敗 |
| 期待値 | - | 1トレード平均損益 | Net Profit / Total Trades | PF |
| シャープレシオ | - | リスク調整後リターン | (リターン-無リスク金利)/σ | ボラティリティ |
| TP Ladder | - | 利確を段階的に設定する手法 | MelodyTrade戦略 | SL/TP |
| SL | ストップロス | 損切ライン | - | TP |
| TP | テイクプロフィット | 利確ライン | - | SL |
| 損益分岐勝率 | - | RR比から逆算した黒字化に必要な勝率 | 1/(1+RR) × 100 | PF, RR比 |

★マーク：週次レビューで頻出する用語

## シート間リンク（GAS管理）

- 統合ダッシュボード → 現物_月次・FX_月次の値を参照
- 週次レビュー → 全シートから集計
- 将来シミュレーション → 現物_月次・FX_月次の過去実績から予測
- 目標vs実績対比 → 統合ダッシュボード・現物_月次・FX_月次の当月値
- 配信者スコア → FX_月次の統計指標
- 経費明細 → 統合ダッシュボードの経費欄に月次合計反映
