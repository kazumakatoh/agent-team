## ① 商品マスター（手入力）

ASIN をキーにして全シートと紐付け。仕入れ価格が変わったらここを更新するだけで全体に反映。

| 列 | 内容 | 入力方法 |
|---|---|---|
| ASIN | 商品識別コード | 手入力 |
| 商品名 | 商品名 | 手入力 |
| 仕入れ単価（円） | 1個あたりの原価（税抜） | 手入力 |
| 仕入れ先 | メーカー・卸業者名 | 手入力 |
| カテゴリ | 商品カテゴリ | 手入力 |
| 備考 | メモ欄 | 手入力 |

---

## ② 日次データ（GAS自動・暫定値）

1行 = 1商品(ASIN) x 1日。毎朝6:00に前日分を自動追記。

| 列 | 内容 | 取得元 |
|---|---|---|
| 日付 | YYYY-MM-DD | 自動 |
| ASIN | 商品コード | Amazon |
| 商品名 | マスター参照 | マスター |
| 売上金額（円） | 税込売上 | SP-API |
| 注文数 | 個数 | SP-API |
| セッション数 | 訪問数 | SP-API (Traffic Report) |
| PV | ページビュー | SP-API (Traffic Report) |
| CVR (%) | ユニットセッション率 | SP-API (Traffic Report) |
| Buy Box率 (%) | Buy Box獲得率 | SP-API (Traffic Report) |
| FBA手数料（円） | 配送・保管料（暫定） | SP-API |
| 返品数 | 個数 | SP-API |
| 返品額（円） | 返金額 | SP-API |
| 広告費（円） | その日の広告費 | Ads API |
| 広告売上（円） | 広告経由売上 | Ads API |
| IMP | インプレッション | Ads API |
| CT | クリック数 | Ads API |
| 仕入れ単価（円） | マスター自動参照 | マスター |
| 仕入れ原価合計（円） | 仕入単価 x 注文数 | 自動計算 |
| ステータス | 暫定 / 確定 | 自動 |

## ③ 経費明細（Settlement Report・確定値）

Settlement Report 到着時に自動取り込み。14日サイクルで確定。

| 列 | 内容 |
|---|---|
| 決済期間 | Settlement の期間 |
| ASIN | 商品コード |
| トランザクション種別 | 注文/返金/サービス料/プロモーション等 |
| 金額（円） | 符号付き金額 |
| 明細種別 | 販売手数料/FBA手数料/保管料/返品手数料等 |

## ④ 商品別収益（GAS自動集計）

集計期間: 当月 / 先月 / 直近30日（切り替え可能）。毎日6:30に更新。

| 列 | 内容 | 計算式 |
|---|---|---|
| ASIN | 商品コード | - |
| 商品名 | - | - |
| 売上合計 | - | SUM |
| 注文数合計 | - | SUM |
| セッション数合計 | - | SUM |
| PV合計 | - | SUM |
| CVR (%) | 平均CVR | 注文数/セッション数 |
| 広告費合計 | - | SUM |
| 広告売上合計 | - | SUM |
| FBA手数料合計 | - | SUM |
| 返品額合計 | - | SUM |
| 仕入れ原価合計 | - | SUM |
| **Amazon内粗利** | **レイヤー1** | 売上-原価-広告費-FBA手数料-返品額-その他Amazon経費 |
| **Amazon内粗利率(%)** | - | 粗利/売上x100 |
| ACOS (%) | 広告費/広告売上x100 | 自動計算 |
| TACOS (%) | 広告費/売上合計x100 | 自動計算 |
| ROAS | 売上合計/広告費 | 自動計算 |
| オーガニック売上(円) | 売上-広告売上 | 自動計算 |
| オーガニック売上シェア(%) | オーガニック売上/売上x100 | 自動計算 |

### 条件付き書式

- Amazon内粗利率 10%以下 → セル赤色
- ACOS 30%超 → セルオレンジ色
- TACOS 30%超 → セル赤色

---

## ⑤ 週次レポート（GAS自動・毎週月曜7:00）

- 直近7日・14日・28日の売上・広告費・粗利の推移表
- 前週比(%)を自動計算
- **グラフ**: 週別売上推移（折れ線・直近8週）
- **グラフ**: 週別粗利推移（棒グラフ・直近8週）

---

## ⑥ 月次レポート（GAS自動・毎月1日 8:00）

- 月別: 売上・広告費・FBA手数料・仕入原価・Amazon内粗利・粗利率
- 前月比・前年同月比(%)
- **グラフ**: 累計粗利（折れ線）
- **グラフ**: 商品別粗利ランキングTOP10（棒グラフ）

---

## ⑦ 広告分析（GAS自動・毎日6:00）

- キャンペーン別: 広告費・売上・ACOS・ROAS・CTR・CPC・CPA
- キーワード別: CT・費用・売上・ACOS
- 検索用語レポート: 成果キーワード・無駄キーワード

### 条件付き書式

- ACOS 30%超 → オレンジ
- CTが多いのに売上ゼロのキーワード → 赤

---

## ⑧ 販売分析（GAS自動・毎週月曜7:00）

- 売れ筋ランキング（注文数順TOP20）
- 死に筋商品（直近60日で注文ゼロ）
- 在庫回転率（売上数/在庫数）
- 曜日別・週別の売上パターン

---

## ⑨ 販促費マスター（手入力・月次）

レイヤー2（最終利益）計算用。社長が月次で金額のみ入力。

| 列 | 内容 | 入力方法 |
|---|---|---|
| 年月 | YYYY-MM | 手入力 |
| Amence 業務委託費 | 44,000円/月 | 手入力 |
| M19 月額固定 | 7,000円/月 | 手入力 |
| M19 成果報酬 | 月額広告費 x 6.5% | 自動計算（広告費連動） |
| 納品人件費 | ママ友バイト実績額 | 手入力 |
| その他 | その他ツール費等 | 手入力 |
| **販促費合計** | - | 自動計算 |
