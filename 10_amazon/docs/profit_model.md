# 利益計算ロジック（2層モデル）

## レイヤー1: Amazon内粗利

Amazonから取得できるデータのみで計算。Settlement Reportの全経費カテゴリを差し引く。

```
売上（税込）
  − 売上原価（商品マスター仕入単価 x 注文数）
  − Amazon販売手数料（Referral Fee）
  − FBA配送代行手数料
  − FBA在庫保管料（月次・長期）
  − FBA関連追加料（返品処理・廃棄・ラベル貼付等）
  − 広告費（Sponsored Products / Brands / Display）
  − 返品・返金額
  − プロモーション費（クーポン等）
  − Amazon月額プラン料金（4,900円/月）
  − その他Settlement Reportに含まれるすべての経費
  ＝ ①Amazon内粗利
```

### 表示場所
- シート④「商品別収益」: 商品ごとのAmazon内粗利・粗利率
- シート⑤「週次レポート」: 週次推移
- シート⑥「月次レポート」: 月次推移・前月比・前年同月比

---

## レイヤー2: 最終利益

Amazon内粗利から、Amazonレポートに含まれない外部経費を差し引く。

```
①Amazon内粗利（全商品合計）
  − Amence 業務委託費（44,000円/月）
  − M19 月額固定費（7,000円/月）
  − M19 成果報酬（月額広告費 x 6.5%）
  − 納品人件費（ママ友バイト実績）
  − その他ツール費用（あれば）
  ＝ ②最終利益
```

### 設計上のルール

- 外部経費は**全商品合計に対してのみ**控除する（商品別按分はしない）
- 入力は社長が月次で「販促費マスター（シート⑨）」に金額のみ手入力
- M19成果報酬は広告費合計から自動計算（入力不要）
- M19運用開始: 2026年4月20日頃（それ以前はゼロで計算）

### 表示場所
- シート⑥「月次レポート」の末尾に「最終利益」セクションとして表示
- 週次Gmail通知にも最終利益を記載
