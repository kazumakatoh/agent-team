## 🛒 物販（Amazon）

### 全商品 月次モニタリング指標

月次レビューでは以下のKPIを商品ごとに確認する。

**全体指標**

| 指標 | 説明 |
|---|---|
| 売上 | 当月の総売上 |
| CV（注文件数） | コンバージョン数 |
| 注文点数 | 販売個数 |
| PV | ページビュー |
| 訪問数 | セッション数 |
| CTR | PV→訪問数比率 |
| CVR | 訪問数→CV比率 |
| 広告費 | 当月広告費合計 |
| TACOS | 総広告費売上比率（全体売上対比） |

| CPA | 顧客獲得単価（点数・件数） |
| 粗利単価 / 利益単価 | 商品ごとの利益水準 |

**広告指標**

| 指標 | 説明 |
|---|---|
| 広告売上 | 広告経由の売上 |
| 広告費 | スポンサー広告費 |
| Perpetua報酬 | 広告コンサル報酬（別計上） |
| IMP | インプレッション数 |
| CT（クリック数） | 広告クリック数 |
| CV | 広告経由コンバージョン |
| CTR | クリック率（CT/IMP） |
| CVR | 広告CV率（CV/CT） |
| CPC | クリック単価 |
| ACOS | 広告費売上比率（広告売上対比） |
| ROAS | 広告ROAS |
| CPA | 広告経由の顧客獲得単価 |

**オーガニック指標**

| 指標 | 説明 |

## 📋 KPIの更新ルール

- **毎月1回**：月次レビュー時に現状欄・各商品テーブルを更新する
- **四半期ごと**：目標値の見直しを行う
- **決算前（1月）**：年間実績の総括と次期目標設定を行う
- **商品撤退基準**：TACOS 30%超・ROAS 3.0未満が3ヶ月連続した場合は要検討

---
*数値は随時更新してください。「-」は未設定または未集計を意味します。*
*2026年途中欄は年度の途中経過のため、通期比較には注意が必要です。*
