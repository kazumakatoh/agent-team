## 🛒 物販（Amazon転売）

### お金の流れ
```
仕入先（各メーカー・卸）
    │
    │ 支払：クレカ or 銀行振込（発注時）
    ▼
自宅（検品・梱包・納品作業）
    │
    │ ヤマト運輸で発送
    ▼
Amazon FBA倉庫
    │
    │ 顧客が注文・購入
    ▼
顧客
    │
    │ Amazon経由で売上発生
    ▼
Amazon（手数料・FBA費用を差し引き）
    │
    │ 2週間ごとに入金
    ▼
会社口座
```

### 主なコスト構造
| コスト項目 | 支払先 | 支払方法 |
|---|---|---|
| 仕入原価 | 各メーカー・卸 | クレカ or 銀行振込 |
| Amazon手数料・FBA費用 | Amazon | 売上から自動控除 |
| 広告費（スポンサー広告） | Amazon / Perpetua | クレカ（広告費×7.5%成果報酬） |
| Amazonコンサル料 | Amence | 請求書 ¥44,000/月 |
| 配送費（FBA納品） | ヤマト運輸 | 口座振替 |

### 差別化ポイント
- 既存カタログに乗らず**独自カタログを作成**→独占販売・高価格維持
- FBA納品は**内製化**（ロジスピ外注から自宅作業に切り替え済み）

## 🆕 YADORIGI（新ブランド）

### お金の流れ（Makuakeフェーズ）
```
製造発注（プロジェクト実施中に並行して進める）
    ├─ ボトル：CKB（中国輸入代行）→ 中国工場
    │       支払：銀行振込・発注時前払い
    │
    └─ 銘木スティック：中野製作所（北海道）
            支払：方法確認中

            ↓ 製造完了

Makuakeプロジェクト公開（2026年8月〜）
    │
    │ 支援者が応援購入
    ▼
Makuake
    │
    │ プロジェクト終了後・末締め翌末払い
    │ 手数料20%を差し引いて入金
    ▼
会社口座
    │
    │ プロジェクト終了後1ヶ月以内に配送
    ▼
支援者（顧客）へ届ける
```

### お金の流れ（一般販売フェーズ・2026年10月〜）
```
製造（CKB・中野製作所）
    │
    ▼
在庫（自宅 or 倉庫）
    │
    ├─ Amazon FBA → 顧客 → 2週間ごと入金
    │
    └─ 自社EC（Shopify）→ 顧客 → 即時入金
```

### 主なコスト構造
| コスト項目 | 支払先 | 支払方法 |
|---|---|---|
| ボトル製造費 | CKB（中国工場） | 銀行振込・前払い |
| 銘木スティック製造費 | 中野製作所 | 確認中 |
| Makuake手数料 | Makuake | 売上から20%控除 |
| デザイン費 | D-noum（藤井さん） | 請求書・都度 |
| D2Cコンサル料 | Reasense（山本さん） | 請求書 ¥33,000/月 |
| Shopify月額費用 | Shopify | クレカ |
| 広告費（Meta・Amazon） | 各媒体 | クレカ |

## 🏠 民泊（和成也ハイム浅草）

### お金の流れ
```
顧客（宿泊者）
    │
    │ 予約・決済（Airbnb / Booking.com）
    ▼
OTAプラットフォーム
    │
    │ チェックアウト後すぐに入金
    │ （OTA手数料を差し引いた金額）
    ▼
会社口座
```

### 主なコスト構造
| コスト項目 | 備考 |
|---|---|
| 家賃 | 固定費（毎月） |
| OTA手数料 | Airbnb・Booking.comへ（売上から控除） |
| 清掃費 | チェックアウトごとに発生 |
| 消耗品費 | アメニティ・トイレットペーパー等 |
| 水道光熱費 | 月次発生 |
| 通信費 | Wi-Fi等 |
| サイトコントローラー | OTA一元管理ツール費用 |

## 📊 会社全体のお金の流れ（サマリー）

```
【収入】
  物販売上（Amazon）─── 2週間ごと入金
  民泊売上（OTA）────── チェックアウト後即入金
  YADORIGI売上────────── Makuake：末締め翌末払い
                         Amazon・EC：都度入金

        ↓ すべて会社口座へ集約

【支出】
  仕入原価（物販・YADORIGI）
  製造費（YADORIGI）
  広告費（Amazon・Meta）
  外注費（Perpetua・Amence・Reasense・D-noum・CKB）
  民泊運営費（家賃・清掃・光熱費等）
  税理士顧問料（¥50,000/月）
  借入返済（西武信金・日本政策金融公庫）
  役員報酬（一馬さん）
```

## 📋 AIへのメモ
- 物販の仕入れはクレカ払いが多く、**キャッシュフローのタイムラグ**に注意（仕入れ→FBA納品→売上入金まで数週間かかる）
- YADORIGIのボトルは**前払い**のため、Makuakeプロジェクト開始前に製造費の資金が必要
- 民泊はチェックアウト後即入金のため、キャッシュフローは安定しやすい
- 総借入約5,900万円の返済スケジュールは月次で資金繰りを確認すること
